# Task Tracker CLI

**Task Tracker CLI** — это простое консольное приложение для управления задачами. С его помощью вы можете отслеживать, что нужно сделать, что уже выполняется и что завершено. Проект позволяет практиковать навыки программирования, включая работу с файловой системой, обработку пользовательского ввода и создание CLI-приложений.

## Возможности

- Добавление, обновление и удаление задач
- Пометка задач как "в процессе" или "выполнено"
- Просмотр всех задач
- Фильтрация задач по статусу:
    - выполненные
    - не выполненные
    - в процессе выполнения

## Структура задачи

Каждая задача имеет следующие свойства:

- **id** — уникальный идентификатор задачи
- **description** — краткое описание задачи
- **status** — статус задачи (`todo`, `in-progress`, `done`)
- **createdAt** — дата и время создания задачи
- **updatedAt** — дата и время последнего обновления задачи

## Установка и запуск

1. Склонируйте репозиторий:
   ```bash
   git clone <URL_репозитория>
   cd task_cli
   go build -o task-cli main.go
   ```
2. Запускайте команду из консоли:
   ```bash
   ./task-cli <команда> [аргументы]
   ```
   
## Пример использования

### Добавление задачи:
   ```bash
   ./task-cli add "Купить продукты"
   ```

### Обновление задачи:
   ```bash
   ./task-cli update 1 "Купить продукты и приготовить ужин"
   ```

### Удаление задачи:
   ```bash
   ./task-cli delete 1
   ```

### Пометка задачи как "в процессе":
   ```bash
   ./task-cli mark-in-progress 1
   ```

### Пометка задачи как "выполнено":
   ```bash
   ./task-cli mark-done 1
   ```

### Просмотр всех задач:
   ```bash
   ./task-cli list
   ```

### Просмотр задач по статусу:
   ```bash
   ./task-cli list-done        # Только выполненные
   ./task-cli list-not-done    # Все, кроме выполненных
   ./task-cli list-in-progress # Только в процессе
   ```
